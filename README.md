# 字符串相似度比较工具

一个纯前端实现的字符串相似度比较工具，支持批量比较两列字符串数据，找到最相似的匹配项。

## 功能特性

### 🎯 核心功能
- **批量相似度比较**：支持比较两列字符串数据
- **多种相似度算法**：编辑距离、Jaro-Winkler等算法
- **实时进度显示**：使用Web Worker后台计算，不阻塞UI
- **结果缓存**：支持增量比较，提高效率

### ⚙️ 比较设置
- **相似度阈值**：可调节相似度阈值（0-100%）
- **文本预处理**：
  - 忽略标点符号
  - 全角转半角
  - 忽略不可见字符
  - 整体字符串比较模式
- **同义词支持**：可定义同义词组，所有词视为同一个
- **权重设置**：可调整不同算法的权重

### 📊 结果展示
- **智能匹配**：显示最相似值和前5个相似值
- **颜色区分**：
  - 黑色：完全相同（100%）
  - 绿色：高相似度（≥80%）
  - 橙色：中等相似度（≥阈值）
  - 红色：低相似度（<阈值）
- **锁定功能**：可锁定满意的结果
- **进度跟踪**：实时显示计算进度

### 💾 数据管理
- **示例数据**：内置示例数据，快速体验
- **数据导出**：
  - 完整结果导出（包含相似度）
  - 简化结果导出（不包含相似度）
- **数据持久化**：自动保存到localStorage
- **数据恢复**：支持加载之前保存的数据

## 使用方法

### 1. 数据输入
- 在"源数据列"中输入被比较的字符串（每行一个）
- 在"目标数据列"中输入比较对象（每行一个）
- 支持从Excel复制粘贴数据

### 2. 设置比较参数
- **相似度阈值**：拖动滑块设置最低相似度要求
- **文本处理选项**：根据需要勾选相应的预处理选项
- **同义词组**：输入同义词，用逗号分隔，如"公司,企业,集团"

### 3. 开始比较
- 点击"开始比较"按钮开始计算
- 观察进度条了解计算进度
- 可随时点击"停止比较"中断计算

### 4. 查看结果
- 结果按相似度从高到低排序
- 点击"锁定"按钮保存满意的匹配结果
- 锁定的结果会优先显示

### 5. 导出数据
- **导出结果**：包含源数据、最相似值、相似度、锁定状态
- **导出简化结果**：不包含相似度信息
- **保存数据**：将当前状态保存到浏览器本地存储

## 相似度算法

### 编辑距离（Levenshtein Distance）
计算两个字符串之间的最小编辑操作次数，包括插入、删除、替换。

### Jaro-Winkler相似度
基于Jaro相似度的改进算法，对前缀匹配给予更高权重。

### 综合评分
结合多种算法的优势，通过加权平均得到最终相似度分数。

## 使用场景

- **数据清洗**：匹配相似的公司名称、产品名称等
- **数据去重**：识别重复或相似的记录
- **数据整合**：合并不同来源的相似数据
- **文本分析**：分析文本相似度
- **质量控制**：检查数据一致性

## 注意事项

1. **数据量限制**：建议单次比较的数据量不超过10000条
2. **浏览器存储**：数据保存在localStorage中，清除浏览器数据会丢失
3. **计算时间**：大量数据比较可能需要较长时间，请耐心等待
4. **内存使用**：超大数据集可能影响浏览器性能
